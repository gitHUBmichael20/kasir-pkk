# Website Kasir Sederhana dengan Tailwind CSS dan Native PHP

Website kasir ini dibangun menggunakan framework CSS Tailwind untuk tampilan antarmuka yang responsif dan menarik, serta bahasa pemrograman PHP natif untuk logika bisnis dan interaksi dengan database.

## Fitur

- Tambah, edit, dan hapus produk
- Kelola kategori produk
- Transaksi penjualan
- Laporan penjualan harian
- Laporan penjualan bulanan
- Pengaturan pengguna

## Teknologi yang Digunakan

- **Frontend**: Tailwind CSS, HTML, JavaScript
- **Backend**: PHP natif
- **Database**: MySQL (contoh)

## Cara Menjalankan

1. **Persiapan:**
   - Pastikan Anda telah menginstall laragon atau xampp

2. **Instalasi:**
   - Clone repositori ini ke direktori web server Anda.
   - Buka terminal atau command prompt di direktori proyek.
   - Jalankan perintah `npm install` untuk menginstal dependensi frontend.

3. **Menjalankan Aplikasi:**
   - Jalankan web server Anda.
   - Buka browser dan arahkan ke direktori proyek di web server Anda.
   - Website kasir siap digunakan.
